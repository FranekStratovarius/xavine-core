# xavine
## linux dependencies:

Gerade werden unter linux außer build-essential keine dependencies benötigt.

### ubuntu (ähnliche):
```
libsdl2-2.0.0
(libsdl2-mixer-2.0.0)
(libsdl2-net-2.0.0)
(libsdl2-ttf-2.0.0)
(libsdl2-gfx-2.0.0)
(libsdl2-image-2.0.0)
```
### manjaro (arch ähnliche):
```
sdl2
(sdl2_mixer)
(sdl2_net)
(sdl2_ttf)
(sdl2_gfx)
(sdl2_image)
```

## debugging
```
coredumpctl gdb
```
